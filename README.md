# cwop.rest

## What is this?

Submitting your weather station data to the NOAA [Citizen's Weather Observer Program](https://en.wikipedia.org/wiki/Citizen_Weather_Observer_Program) is a pain, technologically. Most weather station reporting services use a simple HTTP GET request with URL parameters to submit data. By requiring a direct TCP connection, it is difficult to submit your weather data to CWOP from many contexts. This service assembles an APRS packet and abstracts all of the TCP connection stuff out, so all you have to worry about is one HTTP request.
